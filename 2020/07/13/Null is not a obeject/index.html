Null is not a obeject

null is a object ?

But why there is ‘null is not a obejct’ error?

实际上，这是一个typeof的历史遗留问题，null本身应该是原始数据类型中的一种，而不属于object。

在JavaScript的早期版本中，值以32为存储，其中前3位表示数据类型，其余表示值。对于一个对象，前三位是000，而在JavaScript中，null又表示没有任何内容/无效，因此将其表示为全0，所以才会出现typeof null === 'object'的错误。
