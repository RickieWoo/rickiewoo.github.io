基本数据类型

Posted on 2018-08-06 Edited on 2022-08-09 In keep-learning

String类型（基本数据类型）

转换为字符串

  • toString(baseNumber)方法
    • 数值、字符串、对象、布尔值都有这个方法，参数为输出数值的基数（null和undefined没有该方法）
  • String()方法
    • 不知道操作数是不是null或undefined时，可以用转型函数String()方法，将null和undefined转换为字符串‘null’和‘undefined’

Array类型（引用类型）是Object对象

length属性： 不是只读的，可以用来删除数组的最后几个元素，给它赋值回返回现在的长度（即赋的值）

转换方法

  • toString()

  • valueOf()

  • toLocalString() (对每一个元素执行toLocalString()方法)

  • join() // join(‘,’) = toStirng()

    以上方法若有undefined或者null项则在返回结果中用空字符串表示）

其他方法

  • 栈方法
    • pop()
    • push()
  • 队列方法
    • shift()
    • unshift()
  • 重排方法
    • sort() a在b前返回负数，不变返回0，a在b后返回正数
    • reverse() 反转数组
  • 操作方法
    • concat
    • slice
    • splice(起始位置， 要删除的项数， 要插入的项1， 要插入的项2….)

Object类型（引用类型）

  • 所有对象都有 toLocalString(), toString(), valueOf() 方法，引用对象会重写以上方法

Function 类型

  • 没有重载， 将函数名想象为指针

  • 函数声明和函数表达式

    • 解析器在代码开始之前会读取函数声明，并使其在执行任何代码前可用（被访问）【function sum(){}】
    • 函数表达式必须等到解析器执行到他所在的代码行才会被真正的解释执行 【let sum = function(){}】

  • 函数内部属性

    • arguments（类数组）

      • 主要用途：保存函数参数
      • callee属性，是个指针，指向拥有arguments对象的函数

    • this

      • 函数在执行时所处的作用域

  • 函数属性和方法

    • length
    • prototype
